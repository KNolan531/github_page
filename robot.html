Baseball Robot - Mobile App - Prototype


This is a project I completed for course SWE 4324 - User Centered Design at Kennesaw State University. This project involved teaming up with another student (the user) and using an iterative process to gather requirements and functionality for a mobile app for a robot. Various research, requirements, use cases, and test case documentation are provided below.

The robot is a baseball themed droid that performs various baseball training operations like pitching and hitting. For this project we used Marvel, a prototyping development system to create an interactive app. Through Marvel, the test user can try out the different features of the user interface and give feedback on how they feel about the prototype before implementation.
