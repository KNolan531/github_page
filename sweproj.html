    Pizza Ordering System - C# Prototype


This was a project completed for course SWE 3313 - Software Engineering at Kennesaw State University. The project involved going through the SDLC and ended with a digital prototype. A software design plan, requirements document, paper prototype and test cases were all created. The final prototype with limited features was created using Visual Studio and implemented via C#. Application executable is available upon request.





Project Plan Document

This is the first documentation created for the project. A plan that consists of descriptions of functionality, scope and staff organization. Also a data management and preliminary test plan for the future.



Requirements Document

This is the requirements document compiled for this project. Included are: requirements specifications, use cases, class and er diagrams and a flow of events.



User Guide Document

This was a guide generated to detail the individual interface screens and their purpose. It can serve as an instructional document to help the users testing out the prototype.
