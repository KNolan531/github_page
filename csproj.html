This was an assignment involving creating a basic class to implement the LinkedList data structure with the normal methods and classes/variables involved. Both a Linked List and Node class are used to achieve this. A stack class is also created and extends the linked list class for its implementation.

Array Sorting Time Project (Java)

This was a project I completed for a Discrete Structures course at Kennesaw State University. This project involved running an experiment on 2 different array sorting techniques and then recording the time each completed with various Array Sizes.

Two different classes were created using Select and Bubble Sort techniques and then testing each. Java timer methods are used to keep track of the time and printed out to the user.
